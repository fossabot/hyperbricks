<FRAGMENT>

Category: composite

Type Description

FRAGMENT description

Main Example

fragment = <FRAGMENT>
fragment {
#main
}

Properties

Title Route Section BodyTag Enclose Favicon Template Static Index Meta Doctype HtmlTag Head Css Js
MetaDocDescription

FRAGMENT description

Example

fragment = <FRAGMENT>
fragment {
#main
}
Title

The title of the fragment

Example

fragment = <FRAGMENT>
fragment {
	title = A Fragment Title
}
Route

The route (URL-friendly identifier) for the fragment

Example

fragment = <FRAGMENT>
fragment {
	route = fragment-route
}
Section

The section the fragment belongs to

Example

fragment = <FRAGMENT>
fragment {
	section = some-section
}
BodyTag

Special body wrap with use of |. Please note that this will not work when a fragment.template is configured. In that case, you have to add the bodytag in the template.

Example
no example yet
Enclose

Wrapping property for the fragment

Example

fragment = <FRAGMENT>
fragment {
	enclose = <div>|</div>
}

==== expected output ====
<div></div>
Favicon

Path to the favicon for the fragment

Example

fragment = <FRAGMENT>
fragment {
	favicon = /myfragment.ico
}
Template

Template configurations for rendering the fragment

Example

fragment = <FRAGMENT>
fragment {
	# template
	template {
		template = <<[<div>Fragment: {{value}}</div>]>> 
		isTemplate = true
		values {
			value = some fragment value
		}
	}

}

==== expected output ====
<div>Fragment: some fragment value</div>
Static

Static file path associated with the fragment

Example

fragment = <FRAGMENT>
fragment {
	static = /some/static/path
}
Index

Index number is a sort order option for the fragment menu section. See MENU and MENU_TEMPLATE for further explanation

Example

fragment = <FRAGMENT>
fragment {
	index = 99
}
Meta

Metadata for the fragment, such as descriptions and keywords

Example

fragment = <FRAGMENT>
fragment {

	meta = { 
		description = Fragment description
		keywords = fragment,test
	}
}
Doctype

Doctype for the HTML document

Example

fragment = <FRAGMENT>
fragment {
	doctype = <!DOCTYPE html>
}
HtmlTag

The opening HTML tag with attributes

Example

fragment = <FRAGMENT>
fragment {
	htmltag = <html lang="fr">
}
Head

Configurations for the head section of the fragment

Example

fragment = <FRAGMENT>
fragment {

	head {
		meta {
			charset = UTF-8
		}
	}
}
Css

CSS files associated with the fragment

Example

fragment = <FRAGMENT>
fragment {
	css = [ fragstyles.css, morefrag.css ]
}
Js

JavaScript files associated with the fragment

Example

fragment = <FRAGMENT>
fragment {
	js = [ fragscript.js, anotherfrag.js ]
}

<HYPERMEDIA>

Category: composite

Type Description

HYPERMEDIA description

Main Example

hypermedia = <HYPERMEDIA>
hypermedia {

	title = just a title
	route = index
	section = main
	
}

Properties

Title Route Section BodyTag Enclose Favicon Template Static Index Meta Doctype HtmlTag Head Css Js
MetaDocDescription

HYPERMEDIA description

Example

hypermedia = <HYPERMEDIA>
hypermedia {

	title = just a title
	route = index
	section = main
	
}
Title

The title of the hypermedia site

Example

	hypermedia = <HYPERMEDIA>
	hypermedia {
		title = Welcome
	}
Route

The route (URL-friendly identifier) for the hypermedia

Example

	hypermedia = <HYPERMEDIA>
	hypermedia {
		route = home
	}
Section

The section the hypermedia belongs to. This can be used with the component <MENU> for example.

Example

	hypermedia = <HYPERMEDIA>
	hypermedia {
		section = main
	}
BodyTag

Special body wrap with use of |. Please note that this will not work when a <HYPERMEDIA>.template is configured. In that case, you have to add the bodytag in the template.

Example
no example yet
Enclose

Enclosure of the property for the hypermedia

Example
no example yet
Favicon

Path to the favicon for the hypermedia

Example

	hypermedia = <HYPERMEDIA>
	hypermedia {
		favicon = /favicon.ico
	}
Template

Template configurations for rendering the hypermedia. See <TEMPLATE> for field descriptions.

Example

	hypermedia = <HYPERMEDIA>
	hypermedia {
		template = { 
			template = <<[
				<div>{{myvalue}}</div>
			]>> 
			
			values {
				myvalue = some value
			}
			
			}
	}
Static

Static file path associated with the hypermedia, for rendering out the hypermedia to static files.

Example

	hypermedia = <HYPERMEDIA>
	hypermedia {
		static = /static/path
	}
Index

Index number is a sort order option for the hypermedia defined in the section field. See <MENU> for further explanation and field options

Example
no example yet
Meta

Metadata for the hypermedia, such as descriptions and keywords

Example

		hypermedia = <HYPERMEDIA>
		hypermedia {
			meta = { 
				description = Test meta 
				keywords = test,hypermedia
			}
		}
Doctype

Alternative Doctype for the HTML document

Example

	hypermedia = <HYPERMEDIA>
	hypermedia {
		doctype = <!DOCTYPE html>
	}
HtmlTag

The opening HTML tag with attributes

Example

	hypermedia = <HYPERMEDIA>
	hypermedia {
		htmltag = <html lang="en">
	}
Head

Configurations for the head section of the hypermedia

Example

	hypermedia = <HYPERMEDIA>
	hypermedia {
		head { 
				meta { 
					charset = UTF-8 
				} 
			}
	}
Css

CSS files associated with the hypermedia

Example

	hypermedia = <HYPERMEDIA>
	hypermedia {
		css = [ styles.css, theme.css ]
	}
Js

JavaScript files associated with the hypermedia

Example

	hypermedia = <HYPERMEDIA>
	hypermedia {
		js = [ script.js, app.js ]
	}
